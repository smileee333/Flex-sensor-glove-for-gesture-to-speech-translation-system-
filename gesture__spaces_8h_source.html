1/**
2 * @file gesture_spaces.h
3 * @brief All the declarations for data structures representing gesture space.
4 *
5 * @details
6 * A structure with following fields representing a complete gesture space:
7 * - A variable storing the totol number of gestures present in the gesture space.
8 * - A pointer pointing to an array of gestures.
9 * - A pointer pointing to an array of gesture handlers.
10 * A class declaration representing all permanent gesture spaces and active gesture space present at current time in the system with following data members and member functions:`
11 * - data-members: a pointer pointing to an array of gesture spaces such the each gesture space is represented by the structure defined above, a variable storing first speech gesture space id.
12 * - member-functions: a. Constructor: to initialize all permanent gesture spaces and active gesture space.
13 * b. To search the gesture in a gesture space represented by array id and return the gesture id of corresponding gesture.
14 * c. To invoke the gesture handler corresponding to a gesture defined by it's gesture id with gesture space defined by array id.
15 * d. To invoke the space switch handler responsible for swittching between two gesture spaces (control/speech gesture space) in active gesture space.
16 * e. To extract the gesture string corresponding to a certain gesture represented by it's gesture id along with gesture space represented by array id.
17 *
18 * @author Smile Guleria
