1/**
2 * @file GestureTranslator.h
3 * @brief All the declarations for data structures representing a gesture translator and methods associated with it are defined in this file.
4 *
5 * @details
6 * A gesture representing a word is translated to speech by playing a pre-recorded audio file corresponding to word which is required to be translated. DF mini player module is used to play the audio files.
7 * In SD card insered in DF mini player modules, different folders are created and within each folder different audio files are stored such that each folder represents a gesture space and folder name represents
8 * gestur space id. Similarly, each audio file within a folder represents a word and audio file is named as gesture id. In this way, {gesture id, space id} together represent a unique word and audio file
9 * coressponding to it.
10 *
11 * An instance of DFRobotDFPlayerMini class is created to define a DF mini player module object. It is used to invoke the methods defined in DFRobotDFPlayerMini library to play the audio files.
12 * A generic class meant for interfacing with DF mini player module to enable the smooth playing of audio files corresponding to each word of a sentence one by one. Following data members and member functions
13 * are defined for this class:
14 * data-members: status: It is used to represent the status of DF mini player module being available or unavailable.
15 * member-functions:
16 * *a. constructor: To initialize the DF mini player module object and set the status as available or unavailable based on whether the module is initialized properly or not.
17 * *b. waitUntilAvailable(): To wait until the DF mini player module is available to play another audio file.
18 * *c. isTranslatorAlive(): To return the status of DF mini player module.
19 * *d. translateGesture(): To invoke the right audio file for converting the gesture to speech with gesture id and space id as input.
20 *
21 * @author Smile Guleria
22 * @version 0.0.1
23 *
24 * @copyright Copyright (c) 2023 Smile Guleria
25 * @license MIT License
26 *
27*/
