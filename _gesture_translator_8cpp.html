Detailed Description

This file includes the member function definitions of gesture_translator class defined in GestureTranslator.h file.

This file includes the definition of following member functions of gesture_translator class:

  • a. gesture_translator(): To initialize the status boolen variable representing availability of DF mini player module to translate all the gestures in the captured sentence.
  • b. waitUntilTranslatorAvailable(): To provide the necessary halt in execution of control flow from playing the audio file of next word until previous word's audio file is completely played.
  • c. translateGesture(): To play the audio file relevant to word under consideration.
  • d. isTranslatorAlive(): To return the status boolen variable representing translator being alive or not.

void wakeup_ISR ( )

ISR meant for disabling the sleep mode and also de-attaching the interrupt pin until waiting is required for subsequent audio playback for next word .

This ISR is executed when busy pin of DF mini player module goes high indicating that audio playback is completed.
