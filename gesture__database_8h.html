Detailed Description

All the declarations for data structures representing gesture database.

A structure with following fields represents complete gesture database:

  • A structure instance representing database's meta information
  • An array of structure instances representing database's gesture space's data.
  • An array of structure instances representing database's gesture space's handlers.

A structure declaration representing the structure meant for storing data base's meta information A structure declaration representing the structure meant for storing data base's gesture space's data information. A structure declaration representing the structure meant for storing data base's gesture space's handlers information.
