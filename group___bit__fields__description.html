Detailed Description

Macros for configuration of bitfields for following fields: sensors count, gesture count, gesture space count, words count.


Following bit fields description are used for specifying number of bit fields to be used to store the value of sensors count, gesture count, gesture space count, word count The bit count specified here is used for configuring the maximum allowable count for each above mentioned variables. Thus this bit field description gives designer an additional. tuning parameter over maximum possible value specified by data-type. They should be configured as per requirement.

Note
For example: to have maximum possible numbers of gestures to be 8 within each gesture space gesture space count bits should be kept 3.
