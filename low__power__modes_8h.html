Detailed Description

All the declarations for data structures representing low power modes and methods associated with it are defined in this file.

Typically two low power modes are definied here:

  • SLEEP_MODE_PWR_DOWN : meant for deep sleep during the times when no gesture capture is required as well as during the times when CPU needs to wait for single word translation to complete. CPU wakes-up from this deep sleep mode when start push button is pressed or DF mini player module's busy pin goes high.
  • SLEEP_MODE_IDLE : meant for sleep during the time interval between sampling process completion and wait for next sampling to start. CPU wakes from this sleep mode by simple TIMER interrupt

An enum data type describing power modes supported by cpu

A class meant for checking and configuring the power modes of cpu

  • data-members: mode: it represents the mode in which cpu is currently operating.
  • member-functions: a. To get the current power mode. b. To set the current power mode. c. To start power mode.
