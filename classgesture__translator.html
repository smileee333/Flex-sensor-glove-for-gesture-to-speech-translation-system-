◆ waitUntilTranslatorAvailable()

template<typename max_sensor_cnt_data_type>
void gesture_translator< max_sensor_cnt_data_type >::waitUntilTranslatorAvailable ( )

Method to provide necessary halt in execution of control flow from playing the audio file of next word until previous word's audio file is completely played.

This method checks the busy pin of DF mini player module to know if it is currently playing any audio file or not. If it is playing then it puts the microcontroller in deep sleep mode to save power until the audio file is completely played. Putting microcontroller into deep sleep mode also provide the necessary halt in execution of control flow.
