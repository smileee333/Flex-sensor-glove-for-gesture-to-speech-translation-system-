Detailed Description

All the declarations for data structures representing gesture space.

A structure with following fields representing a complete gesture space:

  • A variable storing the totol number of gestures present in the gesture space.
  • A pointer pointing to an array of gestures.
  • A pointer pointing to an array of gesture handlers. A class declaration representing all permanent gesture spaces and active gesture space present at current time in the system with following data members and member functions:`
  • data-members: a pointer pointing to an array of gesture spaces such the each gesture space is represented by the structure defined above, a variable storing first speech gesture space id.
  • member-functions: a. Constructor: to initialize all permanent gesture spaces and active gesture space. b. To search the gesture in a gesture space represented by array id and return the gesture id of corresponding gesture. c. To invoke the gesture handler corresponding to a gesture defined by it's gesture id with gesture space defined by array id. d. To invoke the space switch handler responsible for swittching between two gesture spaces (control/speech gesture space) in active gesture space. e. To extract the gesture string corresponding to a certain gesture represented by it's gesture id along with gesture space represented by array id.
