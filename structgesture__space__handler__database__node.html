template<typename max_sensor_cnt_data_type>
struct gesture_space_handler_database_node< max_sensor_cnt_data_type >

Gesture data-base's gesture space's handler data-structure.

A structure template with following fields representing gesture database's gesture space's gesture data :

  • cnt : this field is used to store the total number of gesture handlers belonging to a gesture space.
  • handler_ptr : this field is used to store a pointer to an array of dynamically created function pointer pointing to gesture handlers belonging to a gesture space
