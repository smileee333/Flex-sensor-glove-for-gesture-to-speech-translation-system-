Detailed Description

This file includes the member function definitions of sentence_queue class defined in sentences.h file.

This file includes the definition of following member functions of sample_space class:

  • a. addWord(): To add word in the sentence doubly linked list.
  • b. moveNext(): To move the ctrl pointer to next word in the sentence's double linked list.
  • c. movePrevious(): To move the ctrl pointer to previous word in the sentence's double linked list.
  • d. deleteWord(): To delete a word in the captured sentence.
  • e. deleteSentence(): To delete the complete sentence.
  • f. translateGesture(): To invoke the DF mini player module to play the audio files correponding to each captured word in sentence one by one.
