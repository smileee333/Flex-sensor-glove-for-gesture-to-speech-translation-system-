Detailed Description

Macros for configuration of total number of gestures in a gesture space, gesture code and string corrresponding to each gesture in each gesture space.

Gesture codes are defined in the binary format as bxxxxxxxxxx as n bit value where each bit represents a specific sensor output. Thus number of bits in gesture code is defined as per sensor_cnt value. In this project, total 10 sensors are used, so 10 bit values are used to represent gesture code.
