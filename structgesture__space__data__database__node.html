template<typename max_sensor_cnt_data_type>
struct gesture_space_data_database_node< max_sensor_cnt_data_type >

Gesture data-base's gesture space's data data-structure.

A structure template with following fields representing gesture database's gesture space's gesture data :

  • in_gesture: this field is used to store the gesture code meant for entrying into corresponding gesture space.
  • cnt : this field is used to store the total number of gestures belonging to a gesture space
  • gesture_ptr : this field is used to store a pointer to an array of dynamically created gestures belonging to a gesture space
    Note
    for template parameter's and other field's detailed desccripton refer to max_sensor_cnt_data_type_macro, max_gesture_cnt

    max_gesture_cnt for bit-field description
